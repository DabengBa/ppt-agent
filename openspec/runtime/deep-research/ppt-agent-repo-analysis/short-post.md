## 正文

姐妹们（兄弟们）这个项目我研究了一整天，真的被惊到了

ppt-agent 是一个开源的 AI 做 PPT 工具，但它跟市面上那些"一键生成"的完全不一样——

它用了 4 个专业 AI 智能体分工合作：
📋 Research Core — 帮你调研行业资料
📝 Content Core — 用金字塔原理写大纲
🎨 Slide Core — 生成 Bento Grid 布局的 SVG 幻灯片
✅ Review Core — Claude 生成完 Gemini 再审一遍

整个流程分 7 个阶段，其中 3 个需要你确认才往下走
不是那种丢给AI就不管的，是真正的人机协作

看看它生成的小米 SU7 发布会 PPT：
- 封面页：深色科技风 + 品牌橙色一致
- 配置页：4×3 卡片网格，每个配置一张卡
- 对比页：SU7 vs Model 3 vs ET5 数据表格

全部是 SVG 矢量图，放大不模糊，还能 Git 版本控制

几个让我印象深刻的设计：
1️⃣ 每张 PPT 最多放 5 个信息点（Miller 定律）
2️⃣ 5 维评分系统打分，7 分以下自动返工
3️⃣ 10 种 Bento Grid 布局可选
4️⃣ 4 种预设风格（商务/科技/创意/极简）

2778 行代码就搞定了这一切，Markdown 为主，连 npm 依赖都没有

适合人群：开发者、技术演讲者、想要定制化 PPT 的同学
不适合：需要 PowerPoint 格式的商务场景（目前只输出 SVG）

项目地址放评论区，感兴趣的可以试试
