# 素材汇总

- 官方定位：新一代驾驶者之车
- 叙事框架：更好看、更好开、更智能、更安全
- 传播亮点：不仅是配置升级，更是从底层平台、三电、底盘、智驾到安全体系的整体升级

## 3. 四大升级主线素材

### 更好看

- 延续经典比例：车长 4997mm，轴距 3000mm，车宽 1963mm
- 新增 9 款车身颜色，形成跑车 / 时尚 / 豪华 / 经典四大色系
- 外观细节升级：全新进气格栅、更运动化视觉细节、6 款轮毂可选
- 内饰重新设计：对称式中控、金属质感实体按键、三层环绕氛围灯、100% 高频接触区软包

### 更好开

- 全系 V6s Plus 电机，最高转速 22000rpm
- Pro 版 CLTC 续航 902km，Max 版 835km，标准版多家媒体披露为 720km
- 全系碳化硅高压平台：标准版 / Pro 版 752V，Max 版 897V
- Max 版 15 分钟最高补能 670km（CLTC）
- 蛟龙底盘：前双叉臂、后五连杆、固定卡钳、265mm 后宽胎；Pro/Max 提供双腔空簧 + CDC

### 更智能

- 电子电气架构升级：域控四合一，双 5G 双卡双通、Wi-Fi 7、UWB、内置 ETC
- 座舱：后排移动控制屏、6 屏联动、超级小爱、模糊语义导航、车外语音控车
- 辅助驾驶：全系标配激光雷达、4D 毫米波雷达、Thor 平台、700TOPS 算力、小米 HAD
- XLA 认知大模型：从数据驱动走向认知驱动，提升复杂场景判断和绕障能力

### 更安全

- 钢铝混合车身，高强钢和铝合金占比 90.3%
- 新增 2200MPa 小米超强钢、内嵌式防滚架、四门防撞梁升级
- 全系 9 安全气囊，新增后排侧气囊
- 电池包 17 层高压绝缘防护、1500MPa 防刮底横梁、防弹涂层、1230 多项严苛验证
- 门把手三重冗余：车外机械拉手、冗余备份电源、车内应急机械拉手，100% 满足 2027 年新国标要求

## 4. 市场与竞争对比素材

### 对比对象 A：Tesla Model 3

- 起售价：23.55 万元
- CLTC 续航区间：634km - 830km
- 当前公开信息中仍以 15.4 英寸 1080P 中控屏为主，后续或升级 16 英寸 2K 屏
- 价值点：品牌心智强、三电口碑强，但在国内智能驾驶硬件堆料与本地化体验上，小米更激进

### 对比对象 B：比亚迪汉 L EV

- 技术标签：1000V 全域千伏平台、1 兆瓦闪充、5 分钟补能 400 公里
- 价格区间：官方预售 27-35 万元
- 价值点：补能能力极强，是 SU7 在快充维度的重要技术型对手

### 对比对象 C：2026 款小鹏 P7+

- 价格区间：18.68 万 - 19.88 万元
- 纯电续航最高 725km
- 800V 平台、5C 超充、智能化卖点强
- 价值点：更偏价格与智能化竞争，但品牌势能和产品热度未必能压过 SU7

## 5. 建议在讲稿中强调的判断

- 新一代 SU7 的关键不是“参数单项冠军”，而是把设计、性能、智能和安全做成了更完整的综合体
- Pro 版最适合做主推版本，因为 24.99 万价格带兼具 902km 续航和较强配置说服力
- 与 Model 3 对比时要强调“更长续航 + 更强智驾硬件 + 更强中国本地智能体验”
- 与汉 L 对比时要强调“价格带更友好、产品调性更年轻化、品牌热度更强”
- 与小鹏 P7+ 对比时要强调“品牌势能、设计辨识度和发布会传播声量”
