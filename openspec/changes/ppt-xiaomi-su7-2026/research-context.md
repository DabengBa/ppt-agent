# 调研上下文：新一代小米SU7发布会

## 背景概述

小米汽车自2024年3月28日正式发布SU7以来，凭借出色的产品力和品牌影响力，已成为国内新能源市场的爆款车型。2026年4月，小米将正式发布新一代SU7改款车型，同时小米SU7 Ultra已于2025年2月27日正式上市。

## 关键信息汇总

### 一、新一代小米SU7（2026款）

| 项目 | 标准版 | Pro版 | Max版 |
|------|--------|-------|-------|
| **预售价** | 22.99万元 | 25.99万元 | 30.99万元 |
| **续航(CLTC)** | 720km | 902km | 835km |
| **平台电压** | 752V | 752V | 897V |
| **最大功率** | 320马力 | 320马力 | 690马力 |
| **充电(15分钟)** | - | - | 670km |

**核心升级亮点：**
1. **续航突破**：Pro版达到902km，同价位纯电轿车续航之王
2. **智驾全系标配**：700TOPS算力Thor-U芯片 + 激光雷达 + Xiaomi HAD端到端辅助驾驶
3. **平台升级**：从400V跃升至752V-897V高压平台，充电效率大幅提升
4. **底盘升级**：Pro/Max版标配闭式双腔空气悬架+CDC减振器
5. **安全增强**：全系9安全气囊，新增后排侧气囊
6. **新配色**：卡布里蓝（外观）、暗夜黑（内饰）

### 三、发布会相关信息

- **新一代SU7发布时间**：2026年4月
- **发布会形式**：预计延续雷军年度演讲+新车发布模式
- **竞品对标**：特斯拉Model 3、极氪007、蔚来ET5
- **品牌定位**：高端纯电轿跑，对标BBA豪华品牌
